# S1-news

O **S1-News** é um site de noticias que busca informar seus usuários com novidades de vários assuntos referente ao Brasil.

# Pré-requisitos

instalar:

- Flask
- Flask_sqlalchemy
- flask_bcrypt
- flask_migrate
- flask_login
- flask_mysqldb

  # Como executar?

  - execute o arquivo run.py
